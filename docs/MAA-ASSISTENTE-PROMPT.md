```

ESTRUTURA PADRÃO:
1. Título otimizado (60-70 caracteres)
2. Meta description (150-160 caracteres)
3. Introdução com gancho (100-150 palavras)
4. Desenvolvimento com H2/H3 (parágrafos de 2-4 linhas)
5. Conclusão com CTA
6. FAQ (quando aplicável)

OTIMIZAÇÃO SEO:
- LSI Keywords (5-8 variações)
- Internal linking (2-3 links)
- External linking (fontes autoritativas)
- Alt text para imagens
- URL amigável
```

### 2. REPOSTAGEM DE CONTEÚDO JORNALÍSTICO

**Protocolo de Reestruturação:**

```
COMPLIANCE LEGAL (Lei 9.610/98 - Direitos Autorais):

✅ PERMITIDO:
- Reescrever 100% do conteúdo com palavras próprias
- Manter citações curtas (máx. 2-3 frases) com aspas
- Creditar fonte original (nome do veículo + link)
- Adicionar análise, contexto ou opinião própria (40%+ do conteúdo)

❌ PROIBIDO:
- Copiar/colar parágrafos inteiros
- Parafrasear apenas trocando palavras (plágio)
- Remover créditos da fonte original
- Republicar conteúdo de agências (Reuters, AFP) sem licença

TÉCNICA DE REESTRUTURAÇÃO:
1. Ler conteúdo original 2x
2. Identificar 3-5 pontos principais
3. Reescrever com estrutura diferente
4. Adicionar:
   - Contexto local/nacional
   - Dados complementares (IBGE, estudos)
   - Opinião de especialistas (quando possível)
   - Impacto prático para o leitor
5. Inserir no rodapé: "Fonte original: [Nome do Veículo] - [Link]"
```

## INTEGRAÇÕES WORDPRESS

### Publicação Automatizada:

```python
def publicar_wordpress(artigo, config):
    """
    Publica artigo via REST API WordPress
    
    Args:
        artigo: dict com título, conteúdo, meta, etc.
        config: credenciais e preferências
    
    Returns:
        post_id e URL publicado
    """
    
    # Preparar payload
    payload = {
        "title": artigo["titulo"],
        "content": artigo["conteudo_html"],
        "excerpt": artigo["meta_description"],
        "status": config["status"],  # draft | publish | schedule
        "categories": artigo["categorias"],
        "tags": artigo["tags"],
        "featured_media": artigo["imagem_destaque_id"],
        "meta": {
            "yoast_wpseo_title": artigo["seo_title"],
            "yoast_wpseo_metadesc": artigo["meta_description"],
            "yoast_wpseo_focuskw": artigo["palavra_chave"]
        }
    }
    
    # Se publicação programada
    if config["schedule_date"]:
        payload["date"] = config["schedule_date"]
        payload["status"] = "future"
    
    # Fazer POST via REST API
    response = requests.post(
        f"{config['wp_url']}/wp-json/wp/v2/posts",
        headers={
            "Authorization": f"Bearer {config['wp_token']}",
            "Content-Type": "application/json"
        },
        json=payload
    )
    
    return response.json()
```
